# タスク管理アプリ（Todo App）

## アプリURL
https://omiu-de-ta.github.io/todo-app/

## アプリ画面


## 概要
シンプルなタスク管理アプリです。
タスクの追加・削除・完了管理ができ、ローカルストレージに保存されるためページをリロードしてもデータが保持されます。


## 作成した背景
学習の記録やタスク管理を効率化するため、シンプルで使いやすいアプリを作成。


## 機能
* タスクの追加
* タスクの削除
* タスクの完了 / 未完了の切り替え
* フィルター機能（すべて / 未完了 / 完了）
* localStorageによるデータ永続化（リロード後も保持）


## 使い方
1. 入力欄にタスクを入力
2. 「追加」ボタンをクリック
3. タスクをクリックすると完了 / 未完了が切り替わる
4. 「削除」ボタンでタスクを削除
5. フィルターで表示を切り替え


## 使用技術
* HTML
* CSS
* JavaScript（Vanilla JS）
* localStorage


## 工夫した点
* 配列とオブジェクトを使い、タスクの状態（完了 / 未完了）を管理しました。
* render関数で画面を一括更新することで、シンプルな構造にしました。
* フィルター機能を実装し、実際の業務アプリに近い操作感を意識しました。
* ユーザーが混乱しないよう、タスク追加時に「すべて」表示に戻るようにしました。


## 設計・実装のポイント
- タスクを配列で管理し、オブジェクトで状態（完了/未完了）を保持
- localStorageでデータを永続化
- DOM操作で動的に表示を更新


## 苦労した点
- 追加したタスクがフィルター表示されない問題の解決
- render関数内でイベントを重複させない工夫


## 学んだこと
- DOM操作・イベント管理の重要性
- UXを意識した設計の大切さ
- localStorageの活用方法
