# タスク管理アプリ（Todo App）

## 概要

シンプルなタスク管理アプリです。
タスクの追加・削除・完了管理ができ、ローカルストレージに保存されるためページをリロードしてもデータが保持されます。

---

## 機能

* タスクの追加
* タスクの削除
* タスクの完了 / 未完了の切り替え
* フィルター機能（すべて / 未完了 / 完了）
* ローカルストレージによるデータ保存

---

## 使用技術

* HTML
* CSS
* JavaScript（Vanilla JS）
* localStorage

---

## 工夫した点

* 配列とオブジェクトを使い、タスクの状態（完了 / 未完了）を管理しました。
* render関数で画面を一括更新することで、シンプルな構造にしました。
* フィルター機能を実装し、実際の業務アプリに近い操作感を意識しました。
* ユーザーが混乱しないよう、タスク追加時に「すべて」表示に戻るようにしました。

---

## 今後の改善点

* タスクに期限（日付）を追加
* 並び替え機能の実装
* UIの改善（デザイン強化）

---

## 使い方

1. 入力欄にタスクを入力
2. 「追加」ボタンをクリック
3. タスクをクリックすると完了 / 未完了が切り替わる
4. 「削除」ボタンでタスクを削除
5. フィルターで表示を切り替え
